# Apps Script setup

1. Откройте Google Sheets.
2. `Extensions → Apps Script`.
3. Скопируйте содержимое `Code.gs`.
4. Сохраните проект.
5. Запустите `setupArtstudioBooksIntegration()`.
6. Вернитесь в таблицу и обновите страницу.
7. В меню появится `ARTSTUDIO Books`.

GitHub token нужен только для запуска workflow из таблицы. Минимально дайте токену право запускать Actions в конкретном репозитории.
